    Auth Server #

    鉴权 #

    • 请求

      protocol : HTTP method : GET content-Type : application/x-www-form-urlencoded token：Token funcId：功能ID

      eg.

      GET /auth?token=ZW9xpZTdvamE4SWVTNmkK&funcId=10001 HTTP/1.1
Content-Type : application/x-www-form-urlencoded
Host: localhost: 8888
Connection: keep-alive

    • 返回

      根据http state来判断是否鉴权成功；200：鉴权通过 403：鉴权失败 funcSet: 满足条件的funcId集合，以逗号符分割

      eg. 成功：

      HTTP/1.1 200 OK
Date: Wed, 13 Jan 2021 15:27:25 GMT
Content-Type : application/xml
Content-Length: 52

<root><funcSet>100001,100002,100005</funcSet></root>

      eg. 失败：

      HTTP/1.1 403 Forbidden
Date: Wed, 13 Jan 2021 15:27:25 GMT
