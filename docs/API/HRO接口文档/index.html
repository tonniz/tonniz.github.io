    接口设计 #

    报文头 #

    报文头格式为：“<head>报文头内容</head>”，内容结构详见接口设计部分公共报文头

    报文体 #

    报文体主要用于传输业务数据信息，格式为：“<body>报文体内容</body>”

    通讯 #

    Protocol:HTTP Method:POST Content-Type:application/xml;charset=utf-8

    签名 #

    将HTTP Body中的数据流用MD5进行签名，将签名结果与密钥拼接生成签名串，再使用SHA256对签名串进行哈希，结果放在HTTP Request Header中，Key为Signature

    String contentMD5 = byteArrayToHexString(MD5(bodyStream.getBytes("UTF-8")));
String signature  = Base64.encode(SHA256((contentMD5 + secretKey).getBytes("UTF-8")));

    bodyStream 为字节数组

    报文格式 #

    • 请求

      <root>
    <!--报文头-->
    <head>
        <version>v1.0</version>                                             <!--接口版本-->
        <serviceName>diskApply</serviceName>                                <!--服务名-->
        <requestId>20210104000000000001</requestId>                         <!--请求ID-->
        <requestTime>20200104235959</requestTime>                           <!--请求时间--> 
    </head>
    <!--报文体-->
    <body>
        <applyDate>20201229</applyDate>                                     <!--送盘日期-->
        <diskNo>20201229000000000000000000000001</diskNo>               <!--磁盘编号--> 
        <filePath>00001</filePath>    <!--文件名-->
        <fileVersion>v1</fileVersion>                                       <!--文件版本-->
        <customerId>OHJEMIA1EINGUGAH0AITHOHNUEYAHWOO</customerId>           <!--客户ID-->
        <titleId>1000001</titleId>                                          <!--支付地抬头ID-->
        <md5>c5950a259435d3e51262539a10a22910</md5>                         <!--磁盘MD5值--> 
    </body>
</root>

    • 返回

      <root>
    <!--报文头-->
    <head>
        <respCode>000000</respCode>                                         <!--返回吗-->
        <respMessage>交易成功</respMessage>                                 <!--返回信息-->
        <respTime>20210104235959</respTime>                                 <!--返回时间-->
    </head>
</root>

    接口清单 #

    序号 服务名 描述 报文方向 是否签名
    1 diskApply 批量代付申请 HRO -> 沐融代付 Y
    2 diskResult 批量代付结果查询 HRO -> 沐融代付 Y
    3 diskNotify 批次终态通知 沐融代付 -> HRO Y
    4 refundQuery 退票查询 HRO -> 沐融代付 Y
